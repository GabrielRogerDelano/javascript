Manipulando dados com Prompt

OBS 1: Aqui é adicionado o 'Number' para transformar o valor que as variaveis n1 e n2 sejam numeros. Tambem pode adicionar o '.parseInt' ou '.parseFloat' para que seja inteiro ou real

OBS 2: na soma ,se o js não percebese que n1 e n2 sao numeros ele so iria concatenar. (Number + Number ) = adição (String + String) = concatenação

OBS 3: para trasformar em String : ex: var apelido_de_jogo = String(prompt('seu nome no jogo :'))
